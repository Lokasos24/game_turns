## 📖 Descripcion
- El juego tendra la simulacion basica de un juego por turnos, un jugador controlara unidades y las podra mover por el mapa generado, habra enemigos para derrotar y niveles para superar.
- Este juego tambien contara una historia que sera creada sobre la marcha

---

## 🚗 Principales funcionalidades
- ⬜ Seleccion de niveles
- ✅ Mover el personaje seleccionado
- ✅ Restriccion de movimientos
- ⬜ Atacar a un enemigo
- ⬜ Tienda
